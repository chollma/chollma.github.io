Dynamic Pagination

A JavaScript application that given any size list of items, dynamically adds the appropriate pagination links and truncates the list.

Background

I built this project as part of my Fullstack JavaScript Techdegree provided by Treehouse. It utilizes vanilla JavaScript initially display a list of items based on the predetermined pagination threshold. Once the list is displayed, an appropriate number of pagination links are added to the bottom of the page. As the size of the list increases, the pagination will automatically reflect the increase in links.
