Random Quote Generator

A JavaScript application that randomly selects pre-defined quotes from an object and displays them on the screen while changing the color of the container's box shadow.

Background

I built this project as part of my Fullstack JavaScript Techdegree provided by Treehouse. It utilizes vanilla JavaScript to generate a random number, use that number to access an array of quotes and citations. In another function, I used the same random number generator to randomly select a color from an array and change the box shadow of the container.

A quote will appear here in 10 seconds.
