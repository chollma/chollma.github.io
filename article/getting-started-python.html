Getting Started with Python

Let's say that you're a software engineer, data scientist, or just someone who wants to make boring tasks easier and more efficient. Learning Python can be an easy way to automate boring tasks and spend more time on the stuff you like to do.

By Cooper Hollmaier | May 6, 2020, 11:49am PST

I recently started looking into Python to figure out what all the hype was about and I have to say, I'm impressed. This language is easy to use, doesn't have any wild syntax that will stop your code from running, and can help you analyze big chunks of data quickly.

In the example(s) to follow, I'm going to walk you through getting set-up with Python on your machine, understanding the different libraries that functions that are available to you (this way you can choose to build custom functionality or take advantage of hard work that's already been done for you), and show you how to build a small program that will parse the HTML on your pages and provide feedback.

I do want to call out before we get started that Python can make it incredibly easy to parse data, even data that isn't yours. I urge you to resist to desire to "scrape" content and build code that goes against a website's terms of service or just common sense. My goal is to help you do better work on your own website, not steal the great work others are doing.
